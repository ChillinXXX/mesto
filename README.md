# **Mesto. Третья самостоятельная практическая работа.**
***
##  Описание проекта.
Проект является одностраничным WEB-приложением. При разработке использовался компонентный подход и объектно-ориентированная концепция организации кода.   Асинхронный код реализован методом Fetch Promises.  Файловая структура проекта организована по методологии БЭМ.  Для сборки проекта применялся Webpack.

### Функциональность проекта
- Элементы карточек загружаются методом  GET  fetch запроса с backend сервера;
- Данные пользователя загружаются методом GET fetch запроса с backend сервера;
- Реализована возможность добавлять карточки методом POST fetch запроса на backend сервер;
- Реализована возможно удалить карточки методом DELETE fetch запроса на backend сервер;
- Реализована  функционал "лайкнуть" карточк методом PUT fetch запросом на backend сервер
- Реализован просмотр изображения карточки по нажатию на область картинки;
- Реализовано плавное открытие/закрытие попапа с формами и изображением;
- Добавлена валидация форм;
- Добавлена возможность закрыть Popup по клику в области оверлея, нажитию кнопки ESC или по кликукнопки Close.
